Ring 是一个 HTTP 底层库，提供了一套底层的 API，让你可以使用 [Clojure](http://clojure.org/) 来构建 Web 应用。它类似于Ruby 的 [Rack](http://rack.rubyforge.org/) 库，Python 的 [WSGI](http://wsgi.readthedocs.org/)，或者 Java 的 Servlet 规范。
